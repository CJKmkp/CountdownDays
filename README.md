# 桌面倒计日

ICC-CE 插件，用于在桌面悬浮显示倒计时和纪念日。

## 功能

- 支持多个目标事项。
- 支持两种事项类型：
  - **倒计时**：显示距离目标日期还有多少天。
  - **纪念日**：每年自动计算下一次日期，并显示距离下一次纪念日的天数。
- 支持设置标题、备注、目标日期时间和提前提醒天数。
- 提供可拖动的桌面悬浮窗口。
- 可调整窗口透明度、界面缩放、窗口宽度、文字颜色和强调颜色。
- 支持上一项/下一项切换目标。
- 可通过 ICC-CE 通知服务发送临近提醒和到期提醒。
- 配置自动保存，插件重启后保留目标和外观设置。

## 使用方法

1. 在 ICC-CE 插件管理器中安装插件，或下载 Release 中的 `.icpx` 文件并放入 `PluginPackages` 目录。
2. 重启 ICC-CE，插件会自动创建桌面倒计日窗口。
3. 打开 ICC-CE 的插件设置页：
   - 点击“添加”创建目标。
   - 选择“倒计时”或“纪念日”。
   - 设置标题、备注、目标日期时间和提前提醒天数。
   - 使用“编辑”和“删除”维护目标列表。
4. 在设置页调整窗口透明度、界面缩放、文字颜色和强调颜色。
5. 使用“显示 / 隐藏”按钮控制桌面窗口。

## 配置文件

插件配置保存在 ICC-CE 的插件配置目录中：

```text
PluginConfigs/com.icc.countdown-days/config.json
```

配置包括目标列表、窗口位置、窗口外观和已发送提醒记录。修改配置文件前建议先关闭 ICC-CE，并自行备份文件。

## 提醒规则

- “提前提醒天数”控制何时开始提醒。
- 每个目标在同一天只提醒一次。
- 提醒记录保存在插件配置中，最多保留最近 256 条记录。
- 若宿主通知服务不可用，倒计日窗口仍可正常使用。

## 权限

插件声明 `Notifications` 权限，仅用于通过 ICC-CE 通知服务发送临近提醒和到期提醒。

## 构建

需要 Windows 和 .NET 6 SDK：

```powershell
dotnet restore CountdownDays.csproj
dotnet build CountdownDays.csproj -c Release
```

GitHub Actions 支持推送 `v*` 标签或手动运行工作流，并自动构建、打包和发布 `.icpx` 文件。

## 发布包安装

从 GitHub Release 下载：

```text
com.icc.countdown-days.icpx
```

将文件复制到：

```text
PluginPackages/
```

然后重启 ICC-CE，插件管理器会自动处理安装。
